# 📚 Ideology of Pakistan — Interactive Study Guide

> **BS Economics | University of Malakand**
> An interactive, self-paced study guide covering the Ideology of Pakistan for university examinations.

## ✨ Features

- 📖 **2 Parts, 11 Topics** — Full coverage of Theoretical Framework and Historical Context
- 🧪 **Checkpoint Quizzes** — Interactive MCQs after every topic with instant feedback
- ✅ **Progress Tracker** — Mark topics as done, track completion % in real-time
- 🌙 **Dark / Light Mode** — Toggle between themes; auto-detects your system/mobile preference
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- 🔴 **Exam Focus Alerts** — Special callout boxes highlight likely exam questions
- 💾 **Persistent State** — Progress and theme preference saved via `localStorage`

## 🚀 Getting Started

No installation required. Simply open `index.html` in any modern browser:

```bash
# Clone the repository
git clone https://github.com/mustafa-shah-tech/ideology.git

# Open in browser
open index.html
# or on Windows:
start index.html
```

> **Requires an internet connection** for Google Fonts to load correctly.

## 🎨 Design System

The UI uses a CSS custom property system with two complete palettes:

- **Light Mode** — Off-white zinc backgrounds, rich emerald & amber accents
- **Dark Mode** — Deep navy backgrounds, glowing emerald & gold accents

Theme is automatically detected from `prefers-color-scheme` (respects your phone/OS setting) and can be manually toggled with the 🌙/☀️ button in the top bar.
